colorPickerControllerColorChanged:

The color picker will call this method when the color is changing based on live user input.

- (void)colorPickerControllerColorChanged:(UIColor *)color

Parameters

color

The color has been set.

Discussion

The color picker will call this method when the color is changing based on live user input.

This is useful if the application UI needs to update live with user’s input. This callback must be quick.
